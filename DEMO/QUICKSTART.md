# Quick Start - 5 Minutes to Your First Landing Page

Get a landing page running in 5 minutes!

## Step 1: Install (30 seconds)

```bash
pip install snakeskin-xplnhub
```

## Step 2: Create Project (10 seconds)

```bash
snakeskin create my-landing-page
cd my-landing-page
```

## Step 3: Copy the Components

Copy all the component files from the full tutorial into your `src/components/` folder:

- `Navbar.py`
- `Hero.py`
- `Features.py`
- `ContactForm.py`
- `Footer.py`

## Step 4: Update main.py

Replace `main.py` with the code from Step 8 of the full tutorial.

## Step 5: Build & Run (1 minute)

```bash
snakeskin build
snakeskin dev
```

Your browser will open automatically at `http://localhost:3000/dist/index.html`
